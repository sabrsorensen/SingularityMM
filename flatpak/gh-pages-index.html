Singularity Mod Manager

Personal Flatpak repository for Singularity, a No Man's Sky mod manager.

Install (Stable)

Add the repository

flatpak remote-add --user --if-not-exists \
  singularity-mm https://sabrsorensen.github.io/SingularityMM/repo

Install the app

flatpak install --user singularity-mm com.syzzle.Singularity

Update

flatpak update --user com.syzzle.Singularity

Verify GPG signatures (optional)

Import the signing key:

curl -sL https://sabrsorensen.github.io/SingularityMM/repo/flatpak-repo-signing.asc | gpg --import

Then remove and re-add the repository with GPG verification:

flatpak remote-delete --user singularity-mm
flatpak remote-add --user --if-not-exists --gpg-import=flatpak-repo-signing.asc \
  singularity-mm https://sabrsorensen.github.io/SingularityMM/repo

Install (Dev)

PR builds for testing. May be unstable.

flatpak remote-add --user --if-not-exists \
  singularity-mm-dev https://sabrsorensen.github.io/SingularityMM/dev
flatpak install --user singularity-mm-dev com.syzzle.Singularity

Uninstall

flatpak uninstall --user com.syzzle.Singularity
flatpak remote-delete --user singularity-mm
